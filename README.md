# ▶ How to Run

## Clone Repository

```bash
git clone https://github.com/YashSonwane11/Deepskill-project-Cognizant.git
```

---

## Navigate

```bash
cd Deepskill-project-Cognizant
```

---

## Build Project

```bash
mvn clean install
```

---

## Run Spring Boot Application

```bash
mvn spring-boot:run
```

or

Run the main application class from IntelliJ IDEA or Eclipse.
